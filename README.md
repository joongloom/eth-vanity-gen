# eth-vanity-gen

Генератор кастомных красивых адресов в сети Ethereum с использованием Cython для максимальной производительности.

## Standard vs Fast mode

### **Standard Mode**

случайная энтропия => мнемоническая фраза => master seed => приватный ключ

### **Fast Mode**

Прямое получение 32 случайных байт в качестве приватного ключа. Исключает 2048 циклов хеширования для каждой попытки, ускоряя генерацию в сотни раз.

---

### Установка и запуск

Для работы требуется **Python 3.8+** и установленные **C++ Build Tools** .

```bash  
git clone https://github.com/joongloom/eth-vanity-gen.git
cd eth-vanity-gen
pip install -r requirements.txt

# Сборка Cython-модуля (выполняется один раз перед запуском)
python setup.py build_ext --inplace

```

---

#### Параметры запуска:

| **Флаг** | **Описание** |
| --- | --- |
| `-t`, `--threads` | Количество рабочих процессов (воркеров). |
| `-c`, `--console` | Вывод всех проверяемых адресов в консоль. |
| `--fp` | Поиск указанного префикса после 0x. |
| `--fs` | Поиск указанного суффикса. |
| `--fm`, `--fastmode` | Включение режима быстрой генерации. |
| `-j`, `--json` [file] | Сохранение результатов в формате JSON в указанный файл. |
| `--nt`, `--no-text` | Отключение записи в стандартный `results.txt`. |

---

### Перед началом работы изучите проверки в коде `cy_worker.pyx`, раскомментируйте реализованные или добавьте свои. После любого изменения в `.pyx` файле требуется повторная сборка (`setup.py`).

---

### Примеры:

**Поиск префикса cafe на 8 воркерах:**
`python main.py --fp cafe -t 8`

**Fast mode для поиска суффикса beef:**
`python main.py --fm --fs beef`

**Сохранение результатов только в JSON:**
`python main.py -j found.json --nt`
